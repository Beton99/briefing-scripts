# 盘前报告自检清单

在保存报告前，逐项核对以下项目。不符则修正后再保存，修正后仍不符标记为 [自检警告]。

## 可编程自查项

- [ ] 文件名格式：YYYY-MM-DD美股盘前报告.md
- [ ] 抬头第一行包含"> 美股日期："
- [ ] 抬头第二行包含"> 数据源：Yahoo Finance"
- [ ] 抬头第三行包含"> 报告生成："
- [ ] S&P500 行业表 11 行（XLK 到 XLU）
- [ ] AI 产业链表 14 环节全覆盖（1.EDA/IP 到 14.物理AI机器人）
- [ ] 自选股行情表全部标的存在（含非美股 A股5只+港股5只+韩股2只+台股1只+欧股1只，不得删行）
- [ ] 报告不含 TOKEN 计数/用量信息
- [ ] 无"信息覆盖与来源质量"章节（该节仅内部自检用，禁止出现在正文）
- [ ] 报告不含"未出卡标的"表格或清单
- [ ] 代码列已拆分（不写`` `代码<br>*名称*` ``这样的合并格式）

## 内容质量自查项

- [ ] 今日主线写的是"盘前最核心矛盾"，不是流水账
- [ ] 盘前重点每条因果有可点链接
- [ ] 验证锚用"→若X则Y"条件式
- [ ] 自选股异动原因对弱线索写"发现相关线索，但未完成可靠核验"；搜索失败写失败原因
- [ ] 深扫卡所有来源为 `[媒体名](URL)` 格式，无裸 URL
- [ ] 深扫卡每张都有三段（发生了什么/意味着什么/一句话），缺段标注"⚠️ 内容缺失"
- [ ] 深扫卡价格行来自 yf.py 底稿，不来自新闻
- [ ] 财经日历搜当日+未来7日
- [ ] 无交易建议语言（买入/卖出/做多/做空/加仓/减仓/抄底等）
- [ ] 非美股简称按 name_lock.json 锁定表，未自行识别
- [ ] 正常开市未在正文顶部写交易状态说明
- [ ] 免责声明存在
- [ ] 报告正文不含"报告生成时间/数据源/取数成功率"重复行（抬头已有）
- [ ] 35只深扫池全部有终态（有深扫卡=已出卡；无深扫卡=已确认无需出卡）
- [ ] AI 14环节、全球市场、财报目标全部有终态覆盖
