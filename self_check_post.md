# 盘后报告自检清单

在保存报告前，逐项核对以下项目。不符则修正后再保存。

## 可编程自查项

- [ ] 文件名格式：YYYY-MM-DD美股盘后报告.md
- [ ] 抬头第一行包含"> 美股交易日："
- [ ] 抬头第二行包含"> 数据源：Yahoo Finance"
- [ ] 抬头第三行包含"> 报告生成："
- [ ] S&P500 行业表 11 行（XLK 到 XLU）
- [ ] AI 产业链表 14 环节全覆盖（1.EDA/IP 到 14.物理AI机器人）
- [ ] 自选股行情表全部标的存在（含非美股，不得删行）
- [ ] 报告不含 TOKEN 计数/用量信息
- [ ] 无"信息覆盖与来源质量"章节
- [ ] 报告不含"未出卡标的"表格或清单
- [ ] 代码列已拆分（不含 `<br>` 残留）
- [ ] 收盘全景·美股指数表不含前收/前收涨跌/延时价/延时列
- [ ] 收盘全景·全球指数表不含前收/前收涨跌列
- [ ] 大宗代码列已改写为 USD/CNH、AUD/USD、USD/CAD

## 内容质量自查项

- [ ] 已确认美股交易状态；正常收盘时正文顶部未写交易状态说明
- [ ] 休市/半日市已顶部注明（抬头写"半日市，提前收盘"）
- [ ] 未用 quote 旧收盘证明开市
- [ ] 非美简称照锁定表，禁自行识别
- [ ] 深扫卡首条直接写日期和价格数据，不写"状态/价格行："或"价格锚"前缀
- [ ] 无可靠盘后价时未写任何缺失句（"无盘后价/未取到"等全禁）
- [ ] 价格全部来自 yf.py 或行情页，非新闻
- [ ] 无新增/无催化/等待事件驱动者未出卡
- [ ] 全部信号 48 小时内
- [ ] 旧事件最多一条且标"延续交易"
- [ ] 每卡三段齐、每条有可点链接
- [ ] 来源规则已执行：优先来源优先、二线已标[二线来源]、低质已警告且未单独撑主因
- [ ] 禁写"Source:站名"；站名/截图/搜索页不算
- [ ] 第七节自选股包含所有标的，简称照锁定表
- [ ] 第八节盘后异动表仅用 yf.py 延时字段筛选超 ±2%；延时字段缺失/非数字不入选
- [ ] 休市/未确认时未列正常盘后异动表
- [ ] 第八节所有盘后价和盘后涨跌幅均来自 yf.py 延时字段
- [ ] 第九节深扫卡完整三段+全部链接，无缩写改写，仅作原文转载
- [ ] 深扫卡无池外标的、无评分/分层、无宏观复盘
- [ ] 深扫卡不含内部标记 [high]/[medium]/[low]
- [ ] 标题/文件名与美股交易日一致
- [ ] 未用搜索/深扫/新闻价格替代 yf.py 行情
- [ ] 复盘重点/大事记/盘后异动原因未引用新闻盘前/盘后涨跌幅
- [ ] 无交易语言
- [ ] 非美市场日期处理正确；若脚本交易日不同于 market_context 对应日期，正文已标注
- [ ] 最终只新增/更新一个 md（不含中间产物）
- [ ] 已逐只搜索确认 35 只深扫池（每只都有终态，无遗漏）
- [ ] 已区分三种时间（来源发布时间/事件发生时间/本任务交易日）
- [ ] 每张深扫卡主因确实解释当日盘面/盘后
- [ ] 复盘重点含五个子段落（今日主线、核心驱动、跨市场传导、盘后关键变量、次日验证锚）
- [ ] 复盘重点每条因果有可点链接
- [ ] 次日验证锚用"→若X则Y"条件式
- [ ] 收盘全景含三段文字说明（市场宽度与风格、波动与利率传导、财报季）
- [ ] 大事记回扣了昨日关注项（检查昨日简报提到的风险和变量是否兑现）
- [ ] 今日财报用列表格式，覆盖当天盘前+盘后所有重大财报
- [ ] 明日重要财报用列表格式列下一交易日盘前/盘后重要财报
- [ ] 三大指数及 SOX 已做"前收×(1+涨跌)≈今收"自检（自检不写进正文）
- [ ] 2DG.F 已确认 Sivers 或 ISIN=SE0003917798（若未出卡，标注"N/A"）
- [ ] 全球指数历史列完整；AI 14 环节全；S&P500 11 行业完整
- [ ] 免责声明存在
- [ ] 报告正文不含"报告生成时间/数据源/取数成功率"重复行
