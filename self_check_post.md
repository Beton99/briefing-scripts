# 盘后报告自检清单

在保存报告前，逐项核对以下项目。不符则修正后再保存，修正后仍不符标记为 [自检警告]。

## 可编程自查项

- [ ] 文件名格式：YYYY-MM-DD美股盘后报告.md
- [ ] 抬头第一行包含"> 美股交易日："
- [ ] 抬头第二行包含"> 数据源：Yahoo Finance"
- [ ] 抬头第三行包含"> 报告生成："
- [ ] S&P500 行业表 11 行（XLK 到 XLU）
- [ ] AI 产业链表 14 环节全覆盖（1.EDA/IP 到 14.物理AI机器人）
- [ ] 自选股行情表全部标的存在（含非美股 A股5只+港股5只+韩股2只+台股1只+欧股1只，不得删行）
- [ ] 盘后异动表只列 yf.py 延时涨跌幅超 ±2% 的美股标的
- [ ] 报告不含 TOKEN 计数/用量信息
- [ ] 无"信息覆盖与来源质量"章节
- [ ] 报告不含"未出卡标的"表格或清单
- [ ] 代码列已拆分（不写合并格式）

## 内容质量自查项

- [ ] 复盘重点含五个子段落（今日主线、核心驱动、跨市场传导、盘后关键变量、次日验证锚）
- [ ] 复盘重点每条因果有可点链接
- [ ] 次日验证锚用"→若X则Y"条件式
- [ ] 收盘全景含三段文字说明（市场宽度与风格、波动与利率传导、财报季）
- [ ] 板块归因含领涨/领跌/风格切换三段
- [ ] 大事记回扣了昨日关注项
- [ ] 深扫卡所有来源为 `[媒体名](URL)` 格式，无裸 URL
- [ ] 深扫卡每张都有三段（发生了什么/意味着什么/一句话），缺段标注"⚠️ 内容缺失"
- [ ] 深扫卡价格行来自 yf.py 底稿，不来自新闻
- [ ] 今日财报用列表格式，覆盖当天盘前+盘后所有重大财报
- [ ] 明日重要财报用列表格式列下一交易日盘前/盘后重要财报
- [ ] 无交易建议语言（买入/卖出/做多/做空/加仓/减仓/抄底等）
- [ ] 非美股简称按 name_lock.json 锁定表，未自行识别
- [ ] 正常收盘未在正文顶部写交易状态说明
- [ ] 免责声明存在
- [ ] 报告正文不含"报告生成时间/数据源/取数成功率"重复行
- [ ] 35只深扫池全部有终态
- [ ] AI 14环节、全球市场、财报目标全部有终态覆盖
